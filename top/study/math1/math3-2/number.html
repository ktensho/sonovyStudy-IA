二次関数の最大と最小
グラフの問題でよく問われる、最大値、最小値。
二次関数については、どのように考えて求めれば良いか。
今回はあらゆる二次関数の最大値と最小値をササッと求められるようになりましょう。
二次関数の最大値と最小値の問題を考えた時に、大きく２つのパターンがあります。
パターン１は、特にxの範囲、つまり定義域が定められていないパターンです。
このときは、二次関数の頂点を求めることさえ出来れば、簡単に解くことが出来ます。
実際に次の二次関数の最大値と最小値を考えていきましょう。
\(y=a(x-p)^2+q…①\)
今回は二次関数について考えていくので、a=0については考えません。
まずは、二次関数を見たら、頂点と軸を求めましょう。
といってもこの形で式が表されていれば、一目瞭然ですよね。
頂点(p,q) 軸x=p

では、aの範囲を分けて考えていきましょう。
a＞0のとき
考えていきましょうと言っても、①の式を見ただけでは少し分かりにくいですかね？
では、グラフを描いてみましょう。

どうでしょう、グラフを描いてみれば最小値については、何となくイメージついた方も多いのではないでしょうか。
そうです、この二次関数の頂点が最小値です。

イメージとしては、頂点よりも左側から頂点によっていくと、徐々に値が減少していき、
頂点を超えて更に右側へ進んでいくと、逆に徐々に値が増加していきます。
そのため、丁度頂点が最小値となります。
では、最大値についてはどうなるか。こちらについては定義できません。
当然ながら範囲がなければ、どんどん値が大きくなっていき、どこまでも進んでいきます。
なので、最大値については、なし、と答えるしかないです。
a＜0のとき
こちらも同様に、まずグラフを描いて考えてみましょう。

やはり同様に考えていくと、頂点が最大値であることが分かりますね。
イメージも真逆で、頂点よりも左側から頂点によっていくと、徐々に値が増加していき、
頂点を超えて更に右側へ進んでいくと、逆に徐々に値が減少していきます。
そしてこれまた同様に、範囲が決まっていなければ、どんどん小さな値になっていきますし、どこまでも小さな値になっていきます。
そのため、先程と同様に最小値については、なし、となります。
よって、
a＞0　のとき　x=p min q max　なし
a＜0　のとき　x=p max q min　なし
となります。
ちなみに、minは最小を表す、minimumの略で
maxは最大を表す、maximumの略です。
それでは、パターン２を考えてみましょう。
パターン２は、パターン①とは逆に、定義域がある、つまり、xの範囲が決まっている時です。
パターン１は、頂点さえ求めることが出来れば、求められましたが、今回はそう簡単にはいきそうにはないですね。

このパターンについては、いきなり文字のみの例を出すと少し感覚的に掴みにくくなると思いますので
具体例を出して考えていきましょう。
\(y=x^2-6x+4\)
①定義域　(0≦x≦2)

②定義域　(1≦x≦4)
①、②の定義域に関わらず、まずは、おなじみの頂点と軸を求めましょう。
　\(y=x^2-6x+4\)
⇔\(y=(x-3)^2-5\)
よって頂点は、
(3,-5)
軸は、
x=3　
では、まず①の定義域の時

さて、二次関数の最大値と最小値を求める時に、定義域が定められている際は、
まず、いつも通り、頂点と軸を求めた後
軸が定義域の範囲内か範囲外かを考えます。
①のときは、軸は定義域の範囲外ですね。
このときは、上図を見て頂ければ、定義域の端が最大値、もしくは最小値になっていますね。
今回は、
x=0のとき、最大値　4　　
x=2のとき、最小値　-4
このように、軸が含まれていないときは、定義域の端の値を求めれば、最大値と最小値が求めることが出来ます。

それでは、②の定義域の時

この定義域の時は、範囲内に軸が含まれていますね。
このときは、頂点が、最大値もしくは最小値をとります。
このグラフの場合は、最小値をとりますね。
最大値については、定義域の端のどちらか、または、両方となります。
グラフを見て頂ければ、すぐ分かると思いますが、皆さんは一度計算してしっかり確認してくださいね。
x=1のとき、最大値　-1　　
x=3のとき、最小値　-5
パターン２については、他にも軸がaなどの文字で置かれていて、軸が動くような問題もありますが
基本は、まず頂点と軸を求めて、軸が定義域の範囲内か範囲外かを考える。
これを、徹底することにより求めることが出来ますので、あらゆる問題に触れて練習していきましょう。
さて、話は一転して二次方程式についてお話していきます。
今まで様々な方程式を学んできたかと思いますので、何となく二次方程式がどのようなものか予測がつくかと思いますが
まずは、定義から述べさせて頂きます。
式を整理して、(2次式)=0　の形に変形できる方程式を、２次方程式といいます。

一般的に、xの二次方程式は
\(ax^2+bx+c=0\)
と表されます。ただし、a≠0　とします。
ここで、a≠0とする理由は、単純にa=0だと、
一次方程式になるためです。
意外とこれが忘れがちになるので、しっかり覚えておいてくださいね。
何事もなく、二次方程式について説明始めましたが、
急に二次関数の話から、何故二次方程式の話をしだしたのか疑問に思われている方もいらっしゃるかもしれませんね。
しかし今まで、一次方程式を学んできた中で、グラフを使って問題を解いてきませんでしたか？
もしも、一次方程式、特に連立方程式をグラフを使って解いたことがなければ、一度解いてみてください。
というよりは、２つの一次方程式のグラフの交点を求める問題が、連立方程式の問題と見ることはできませんか？
このように、方程式と、関数やグラフは、密接な関係にあります。
二次方程式については、二次関数をある程度学ぶとよりわかりやすくなる点があることと、
逆に二次方程式について学んでいないと、二次関数で解きにくい問題があるので
このタイミングで二次方程式について紹介させて頂くことにしました。
それでは、二次方程式の解き方ですが、主に２つあります。
まず１つが、因数分解を使って解いていく方法です。
１つ具体例を出しましょう。
\(x^+x-6=0\)
左辺の因数分解は、もう皆さんお手の物ですよね？
ちょっとつまずく方は、二次方程式の分野では、よく使われますし、他の分野でも頻出なので、
因数分解の練習をもう少しやっておきましょう。
では、左辺について因数分解をして
　\(x^2+x-6=0\)
⇔\((x+3)(x-2)=0\)
さて、xがどのような値をとれば、左辺の式が、0になるでしょうか。
実はこれをわかりやすくするために、因数分解を行いました。
左辺の、(x+3)、または、(x-2)、の部分が、0になれば
左辺が、0になることがお分かりでしょうか。
分かりにくい方は、理解するために、次のように文字を置いてみましょう。
A=(x+3)
B=(x-2)
こうすれば、元の式は次のように表せますよね。
AB=0
この形であれば、AまたはBが、0になればいいことがわかりやすくなりましたよね。
よって、A=0またはB=0、つまり、(x+3)=0または(x-2)=0　を求めれば、この方程式の解が求められます。
　(x+3)=0
⇔x=-3
　(x-2)=0
⇔x=2
ゆえに、\(x^2+x-6=0\)　の解は、x=-3,2 となります。
この解き方のいいところは、因数分解さえしてしまえば、すぐに解がわかるところです。
そして、この解き方を利用すれば、次のような二次関数の問題が同様に解くことができます。
次の二次関数のグラフと、x軸との交点の座標を求めよ。
\(y=x^2+x-6\)

さて、いつもであれば、頂点と軸を出しましょうと言うところですが、
この問題に限り、すぐに因数分解が分かったときは、軸などを求める前に因数分解をしてしまいましょう。
　\(y=x^2+x-6\)
⇔\(y=(x+3)(x-2)\)
x軸と交わる、つまりy=0なので
\((x+3)(x-2)=0\)
こちらの解を求めることになります。
これは、先程解いた二次方程式と同じですね。
ここまで言えば答えについては、皆さん自身の手で、問題なく求められると思いますので省略しますね。
この問題を見た後ですと、二次方程式の問題は、二次関数のグラフとx軸の交点を求める問題は、
同じことであることは、ご理解いただけましたでしょうか？
少なくとも、二次方程式と二次関数が無関係ではないことはお分かり頂けたと思います。
さて、では次のような二次方程式の問題のように因数分解が出来ないようなものや、
因数分解は出来そうだけど、なかなか因数分解出来ずに困った時はどうしましょうか。
\(ax^2+bx+c=0\)　(ただし、a≠0)
こんな問題見せられて、因数分解しろって言われても出来ないですよね…
実は因数分解をしなくても、二次方程式の解を一発で求めることが出来る公式があります。
そんな便利な公式があるなら、先に教えろよ！とか、因数分解なんて出来なくても問題ないじゃないか。
なんて思われるかもしれませんが、そういいことばかりでもないです。
因数分解がすぐ出来る問題であれば、紹介する公式を使うよりも、因数分解で解いた方が圧倒的に速く答えに辿り着くことが出来ます。
そのため、今回のように因数分解が明らかに出来ない時や、因数分解がうまく出来ない時、
あとは、検算するときに使うようにしましょう。
今回は公式を紹介する前に、導いていきます。
\(ax^2+bx+c=0\) (ただし、a≠0)
この式は、次の二次関数のグラフと、x軸の交点とみることができますよね。

式で分けて表すと、次の２式で表します。
\(y=ax^2+bx+c\)
y=0
さて、先程の因数分解を利用した時の例では、求めませんでしたが
二次関数といえば、頂点と軸を求めましょう。
そしてこの計算少し大変かもしれませんが、
面倒くさがらずに、計算練習のついでにやってみましょう！
　 \(y = ax^2 + bx + c\)
⇔ \(y = a( x+ b/2a )^2 - b^2/4a + c\)
ちなみに、今更ですが、この軸と頂点を求めているときにしている計算を平方完成といいます。
よく使われる言葉ですので、覚えておきましょう。
ここで、y=0　を代入してみましょう。
ついでに、見やすくするため、左辺と右辺を入れ替えます。
\(a( x+ b/2a )^2 - b^2/4a + c　= 0\)
⇔ \(a( x+ b/2a )^2 = b^2/4a - c\)　
⇔ \(a( x+ b/2a )^2 = (b^2　- 4ac)/4a\)
⇔ \(( x+ b/2a )^2 = (b^2　- 4ac)/4a^2\)
さて、ここで両辺の平方根をとる時に、
本来場合分けするべき点がありますが、スムーズに説明するために一旦省きます。
皆さんは私が説明するまでの間、何を場合分けしなければいけないのか考えてみてください。
⇔ \(x+ b/2a = ±√(b^2　- 4ac)/√(2a^2) \)
⇔ \(x = -b±√(b^2　- 4ac)/2a\)
長い道のりでしたが、解が出ました。
そして、この解の形が、解の公式と呼ばれています。
a、b、cに、それぞれ該当する値を代入すると、解が求められます。
最悪この導いた流れだけでも、完璧にしておけば求められますが、時間かかるのは目に見えていますよね。
複雑な公式で、代入して計算するのも少し大変かもしれませんが
困ったときに役立ちますので、覚えておきましょう。
と、公式をスムーズに紹介するために
後回しにした場合分けについて考えましょう。
皆さん何を場合分けしないといけないか分かりましたか？
ここで一番無視出来ない場合分けは、「b^2-4ac\)」の大きさについてです。
ルートの中が負の値になってはいけませんよね。
それでは考えていきましょう。

\(b^2 - 4ac ＞ 0 \)のとき
このときは問題ないですよね？解が２つあり、
グラフで表すと下図のように、x軸と2点で交わるグラフになります。

\(b^2-4ac=0\)のとき
このときは少し特殊で、解が１つになるときになります。
一例を出すと、次のような二次方程式ですかね。
　\(x^2 - 2x + 1 = 0\)
⇔ \((x-1)^2 = 0 \)
このように、因数分解すると、(px-q)^2=0 の形になるものです。
また、二次方程式の解が１つの時は、重解を持つといいます。
そして、グラフにすると、下図のように、x軸に１点で接します。


\(b^2-4ac＜0のとき\)
さて、問題の負の値をとるときです。
ここまで、場合分けしてきて、わざわざグラフも描いている点から、察しのいい方はお気付きでしょう。
今回に関しては１つ例を出して先にグラフを見て頂いた方が、実感しやすいですかね？
\(x^2 - x + 1 = 0\)
　\(b^2-4ac　= 1-4\)
⇔\(b^2-4ac　= -3\)
と、見ての通り、負の値になりました。
では、このときのグラフを描いてみましょう。

見ての通り先程までのグラフとは違い、x軸と交わることも接することもありませんね。
つまり、どういうことか。この二次方程式には、実数の範囲では、解が存在しません。
二次方程式だけで、考えていると、解の公式とまで説明されているにも関わらず
ルートの中が負の値のものは、解がないとはどういうことなのか、分かりにくいのですが、
このようにグラフと関連させて考えると、納得は頂けるのではないでしょうか。
そりゃx軸と交わらない、つまり、「y=0」　なんて存在しないにも関わらず、
y=0を求めさせるような二次方程式を前にしたら、
困った時の解の公式でもお手上げでしょう？
なので、ルートの中を負の値にして、この二次方程式には解がないぞと伝えてくれているのですね。
流石、困った時の解の公式です。解がないときは、きっちり一目で分かるように教えてくれるのです。
と、少し私の勝手なイメージで解の公式に、感情移入してみましたが
ちょっと苦手な問題や、公式と出会った時は、自分なりにストーリーを設定してみるのも面白いかもしれません。
もしもなかなか出来ない問題や、なかなかしっくり来ない公式があるときは一度試してみてください。
もちろん合う合わないもありますから、無理にはしなくて大丈夫ですよ。
話が長くなりましたね、(\b^2-4ac＜0\)のときは、「解なし」が答えです。
そして、この、「\(b^2-4ac\)」は、「\(D=b^2-4ac\)」とよく表され、判別式と呼ばれています。
a、b、c、の値さえ分かれば、その二次方程式の解があるのかないのか、
解があるなら、２つなのか、１つなのかを判別出来ますからね。
おそらくこの、「D」も「Discrimination」という判別という意味のある英単語から来ているのではないでしょうか。
このように数学の式に、何か文字を置くときは、英単語の頭文字などから取ってきていることがよくあります。
私も、できる限り紹介していきますが、意味がありそうな文字の置き方をしてるのではないかと感じたとき、
少し調べてみてください。どういう意味でその文字が置かれているのか、理解したり、認識したときに
その式の表す意味が分かってくるので、使いどころがわかりやすくなります。
細かいかもしれませんが、色んなところにアンテナを張りつつ学習していきましょう。
