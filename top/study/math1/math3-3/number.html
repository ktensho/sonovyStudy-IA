二次不等式
方程式を学んだ後は、不等式について考えていきましょう。
さて、二次不等式とは、どのようなものでしょうか。
まずは、式で表してみましょう。
\(ax^2 + bx +c ＞ 0\)
\(ax^2 + bx +c ＜ 0\)
このようなに、表せる不等式を、二次不等式と言います。
そして二次不等式も、二次関数のグラフを描くことにより、求めやすくなります。
慣れれば計算だけで求められるようになりますが、イメージしやすいので、
二次不等式の問題で上手く答えを求められない時は、グラフを描いてみることを忘れないようにしましょう。
それでは、グラフを利用して不等式を解いていきますが、
このとき役に立つのが、判別式です。
例を見ていきましょう。

\(y = ax^2 + bx +c \)
\(D= b^2 - 4ac\)
このときの、判別式と、aの範囲(a≠0)によって、
y ＞ 0　または、y ＜ 0 どちらになるのか、考えます。
D＞0　のとき、
このとき、二次関数は、x軸と交点が２点あります。
この二点を、α、β、とします。(α＜β)
少し注意していただきたのですが、この交点αと、aの文字はよく似ていますが違う文字です。
今後もよく使われる文字なので、違う文字であることをしっかり認識しておいてください。
さらに、a＞0のとき、

二次不等式を解くとき、グラフの使い方ですが、
x軸よりも、下の部分にある範囲は、y ＜ 0　
x軸よりも、上の部分にある範囲は、y ＞ 0　
当たり前のことですが、このことさえしっかり覚えておけば、
二次不等式については、グラフを描けば簡単に解くことができます。
それでは、上図より
D＞0　かつ　a＞0　のとき、
x＜α、または、β＜x　のとき、y＞0
α＜x＜β　のとき、y＜0
つまり、この範囲のときに、
\(ax^2 + bx +c ＜ 0\)
といった問題を出された場合は、
α＜x＜β
これが、答えるべきxの範囲です。
次に、D＞0、かつ、a＜0　のとき

同様に考えると、
x＜α、または、β＜x　のとき、y＜0
α＜x＜β　のとき、y＞0
D＞0　の範囲のときは、以上です。
解き方さえ分かれば簡単なものでしょう？
D=0 かつ　a＞0　のとき、

D=0　のときは、グラフの通り、
二次関数のグラフは、x軸の１点に接します。

このときの、解を、ｐとすると、
x≠p のとき、y＞0
y＜0　は、範囲なし
これもグラフを見て頂ければ、この範囲のときは、
x軸よりも下の値は、取らないことがお分かり頂けると思います。
このグラフの頂点、つまり最小値が、0のため、負の値を取らないのは当然ですよね。
D=0 かつ　a＜0　のとき、

同様に考えて、
x≠p のとき、y＜0
y＞0　は、範囲なし
D＜0　かつ　a＞0　のとき
(二次不等式　図５)
この範囲のときは、x軸と交わることはありません。
ある意味一番,二次不等式ではわかりやすいかもしれません。
すべてのxにおいて、y＞0
y＜0　は、範囲なし
D＜0　かつ　a＜0　のとき
(二次不等式　図６)
同様に考えて、
すべてのxにおいて、y＜0
y＞0　は、範囲なし
いかがでしたでしょうか。
二次不等式についての説明は少なかったかもしれませんが、
逆にほんの少しの事柄を、きっちり覚えることにより簡単に問題を解ける分野であると思っています。
特に単純な二次不等式の問題であれば、今回の内容のみで網羅されています。
グラフを描けば確実に解けますが、グラフを描かなくても、そこまで難しくはないので
すぐに解けるぐらいまで、しっかり類題を解いて学習していきましょう。
